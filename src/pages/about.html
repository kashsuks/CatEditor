About Pinel

Pinel is a blazingly fast code editor built with Rust, designed for speed and simplicity.

Our mission is to provide developers with a lightweight, high-performance editor that gets out of your way and lets you focus on what matters — writing code.

Why Pinel?

  • Fast — Built from the ground up in Rust for maximum performance
  • Simple — Minimal configuration, maximum productivity
  • Portable — Works on macOS, Linux, and Windows

Get Involved

Pinel is open source. Check out the GitHub repository to contribute or report issues.
